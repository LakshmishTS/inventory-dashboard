# Inventory Performance Dashboard

![inventory-dashboard](Screenshot 2025-12-09 100122.png)


An **end-to-end inventory analytics project** built using **Excel and Power BI**.  
This project focuses on tracking stock value, low-stock items, overstocking, and supplier performance to support data-driven inventory decisions.

---

## Tools & Technologies

- **Excel** – Data storage and initial cleaning  
- **Power BI** – Data modeling, DAX measures, and interactive dashboard  
- **Git & GitHub** – Version control and project documentation  

---

## Key Insights

- Monitors **total stock value, total products, categories, and current stock** in one consolidated view.  
- Highlights **items below reorder level** and **overstocked products**, helping balance inventory and reduce holding or stockout risk.  
- Reveals **which categories and suppliers contribute most to inventory**, and identifies **top products by stock value** and those **below reorder level** for timely action.
